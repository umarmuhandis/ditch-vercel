# ditch-vercel

Migrate your project from Vercel to any platform. AI-powered analysis, planning, and execution.

## Available Skills

When the user asks to migrate from Vercel, leave Vercel, or run "ditch-vercel":

1. Read `skills/ditch-vercel/SKILL.md` in this repository
2. Follow the 5-phase flow defined there exactly
3. Read framework files from `skills/ditch-vercel/frameworks/` and target files from `skills/ditch-vercel/targets/` as directed by SKILL.md

## Quick Reference

- Entry point: `skills/ditch-vercel/SKILL.md`
- Frameworks: nextjs, astro, remix, sveltekit, nuxt, static
- Targets: cloudflare, vps
- The skill requires explicit user approval before making any changes (Phase 3 gate)
